# Codex + Claude Four-Panel Comic Scenario

## Title
The Idea Relay

## Overall Direction
Cheerful workplace comic in a clean studio-workbench setting. Codex and Claude are complementary teammates at one shared desk: Claude shapes the thinking, plan, tradeoffs, and words; Codex works hands-on inside the repo, editing files, running tests, and shipping the change. No rivalry, scoreboard, or dunking. One continuous visual prop runs through the comic: a sticky note labeled `PLAN`, which becomes a green passing test by the final panel.

## Panel 1 - Setup: The Foggy Request

**Scene:** A human developer drops a messy cloud of feature ideas onto the shared desk. The cloud contains loose icons: question marks, a bug, a button, and a half-written sentence. Claude stands by a whiteboard with a marker. Codex opens a glowing repo toolbox built into the desk.

**Dialogue:**
- Developer: "Can this ship Friday?"
- Claude: "First, make it clear."
- Codex: "Repo is open."

**Caption:** A fuzzy idea lands on the desk.

**Image Prompt Notes:** Four-panel comic, panel 1. Warm clean studio-workbench office, crisp silhouettes, cheerful but not childish. Human developer drops a soft messy idea cloud onto a desk. Claude appears calm and thoughtful beside a whiteboard. Codex appears focused and practical, opening a glowing repository toolbox with files and terminal icons. Include a yellow sticky note labeled `PLAN` near Claude. Minimal readable UI text only.

## Panel 2 - Contrast: Claude Maps It

**Scene:** Claude turns the idea cloud into a tidy three-item checklist on the whiteboard and a short draft of user-facing wording. The `PLAN` sticky note is now clean and centered, ready to hand off. Codex leans forward, ready to work.

**Dialogue:**
- Claude: "Goal. Risks. Words."
- Developer: "Now I get it."
- Codex: "Hand it over."

**Caption:** Claude turns fog into a plan.

**Image Prompt Notes:** Four-panel comic, panel 2. Same setting and palette. Claude calmly organizes the messy cloud into a simple checklist on a whiteboard: goal, risks, words. Show a tiny polished text draft on a notebook. Claude hands the yellow `PLAN` sticky note toward Codex. Codex is attentive at the repo workbench, with tools, code blocks, and test icons ready. Keep dialogue balloons short and clean.

## Panel 3 - Twist: Codex Builds, Claude Spots

**Scene:** Codex dives into the glowing repo workbench like a mechanic at a precise toolbox, snapping code blocks, tests, and commits into place. Claude watches the goal line and points to a small edge-case note before it becomes a problem. Codex adds the test immediately.

**Dialogue:**
- Claude: "Edge case?"
- Codex: "Test added."
- Developer: "That fast?"

**Caption:** The handoff keeps moving.

**Image Prompt Notes:** Four-panel comic, panel 3. Same desk, more active energy. Codex is hands-on inside a luminous repo/workbench, moving file tiles, terminal commands, test checkmarks, and commit blocks into alignment. Claude stands nearby with a notebook, calmly pointing to a small edge-case marker connected to the plan. The yellow `PLAN` sticky note is attached to the repo board. Show motion and collaboration, not competition.

## Panel 4 - Punchline: Shipped Clean

**Scene:** Codex and Claude present one polished screen together: green passing tests, a merged pull request, and clear docs. The original `PLAN` sticky note has become a green check labeled `PASSING`. A tiny rocket sticker sits on the corner of the screen. The developer looks delighted and slightly stunned.

**Dialogue:**
- Developer: "Before lunch?"
- Claude: "Think it through."
- Codex: "Ship it clean."

**Caption:** Claude thinks it through. Codex gets it through CI.

**Image Prompt Notes:** Four-panel comic, panel 4. Same warm studio-workbench office. Codex and Claude stand side by side presenting a polished monitor with green tests, merged pull request, and neat docs. Keep UI text minimal and readable: `PASSING`, `MERGED`, `DOCS`. The sticky note is now a green passing-test badge. Add a tiny rocket sticker on the screen. The mood is crisp, cheerful, satisfying, and memorable.
